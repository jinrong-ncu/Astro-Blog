刚上大学的时候，很多人买电脑后的第一件事不是学习，而是到处找“装机必备”。问题是，软件一多反而会乱：笔记一个工具，文档一个工具，代码一个工具，文件又散在几个网盘里。最后工具装了一堆，真正能坚持用下来的没几个。

这篇文章不追求把所有学生优惠都列满，而是整理一套比较稳的免费工具栈。它们大多有官方免费版本，部分工具对学生或教育邮箱提供额外权益。你可以按自己的专业和使用场景挑，不需要一次性全部装上。

## 写作和文档

最基础的选择是 **Microsoft 365 Education** 和 **Google Docs**。

如果学校已经开通 Microsoft 365 Education，学生通常可以用学校邮箱登录，使用 Word、Excel、PowerPoint、OneNote、Teams 等工具。不同学校开通的计划不完全一样，有的只提供网页版本，有的会包含桌面版授权，所以最稳妥的方式是先去学校信息化平台或 Microsoft Education 页面检查邮箱资格。

Google Docs 的优势是协作轻，适合小组作业、课堂展示和快速共享文档。它不一定依赖 edu 邮箱，普通 Google 账号也能用，但在国内网络环境下可能不如 Microsoft 365 稳定。

如果你经常写论文、课程报告或项目文档，我建议文档类工具不要频繁更换。真正重要的是形成固定流程：资料收集、草稿、正式文档、版本备份，而不是今天用一个模板，明天换一个平台。

## 笔记和知识整理

**Notion** 适合做课程主页、读书笔记、项目管理和个人知识库。它的教育计划面向符合条件的高校学生和教育工作者，通常需要使用学校邮箱登录或升级。Notion 的强项是结构灵活，可以把课程、任务、阅读材料和项目进度放进同一个空间。

如果你更喜欢本地、长期可控的笔记方式，可以试试 **Obsidian**。它的基础功能免费，文件以 Markdown 形式保存在本地，适合长期沉淀知识。缺点是刚开始需要自己搭结构，不如 Notion 开箱即用。

我的建议是：课程管理用 Notion，长期知识沉淀用 Obsidian。前者像一个学习工作台，后者像一个私人资料库。

## 编程和开发

学编程最值得先申请的是 **GitHub Student Developer Pack**。它不是一个单独的软件，而是一组学生开发者权益，包含 GitHub 自身权益和一些第三方工具资源。通过后，你可以围绕 GitHub 建立自己的项目主页、代码仓库和作品集。

IDE 方面，**Visual Studio Code** 对大多数前端、Python、数据分析入门已经足够。它免费、插件多、社区资料丰富。对于 Java、Kotlin、Python、PHP、C# 等专业开发，**JetBrains Student Pack** 很值得申请，学生和教师可以免费使用 JetBrains 的专业 IDE，但教育许可证不能用于商业工作。

如果你只是刚开始学编程，不要把时间花在“哪个编辑器最好”上。先选一个稳定工具，把环境配好，然后坚持写项目。

## 设计和原型

**Figma for Education** 是设计、产品和前端同学都值得了解的工具。Figma 本身有免费 Starter 计划，符合条件的学生和教育工作者还可以申请 Education 计划。对于高校学生，Figma 官方说明通常对应 Professional 计划；高中学生和教育工作者可能对应 Enterprise 计划，具体以官方验证结果为准。

Figma 不只是 UI 设计软件。你可以用它做课程展示图、产品原型、小程序页面草图、海报、简历版式，甚至和同学一起在线协作。

如果你只是做普通海报或社媒图，Canva 的免费版也够用。需要注意的是，Canva Education 更偏向 K-12 教育场景，不一定适合所有大学生个人申请。

## 云服务和个人网站

想做个人作品集或课程项目，可以从 **GitHub Pages**、**Vercel**、**Cloudflare Pages** 这些免费部署平台开始。它们很适合静态博客、前端项目和作品集页面。

我现在这个博客就是典型的个人网站路线：内容用 Markdown 写，框架用 Astro，部署后可以接入 RSS、站点地图、搜索和统计。对学生来说，这比只把项目放在本地电脑里更有价值。一个长期维护的网站，就是你的公开作品集。

## 怎么选择

如果你不知道从哪里开始，可以按这个顺序：

1. 文档：Microsoft 365 Education 或 Google Docs
2. 笔记：Notion 或 Obsidian
3. 编程：VS Code + GitHub
4. 专业 IDE：JetBrains Student Pack
5. 设计协作：Figma
6. 展示作品：GitHub Pages 或 Vercel

学生软件福利最容易踩的坑，是把它当成“免费会员收集游戏”。其实更好的用法是搭一套稳定的学习系统：文档有地方写，代码有地方放，笔记能长期找回，作品能公开展示。工具越稳定，你越能把注意力放回学习和创作本身。

> 说明：各平台的教育权益会调整，申请前建议以官方页面为准，尤其是地区限制、学校邮箱识别和商业用途限制。
