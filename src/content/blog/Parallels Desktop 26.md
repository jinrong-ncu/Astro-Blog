## 🚀 安装步骤 (保姆级)

### 第一步：安装主程序
双击下载好的 `.iso` 文件，然后按照提示安装即可。

**注意**：第一次打开时，如果系统提示“无法验证开发者”或者“应用已损坏”，去「系统设置」->「隐私与安全性」里点一下“仍要打开”就行。

### 第二步：解决“网络初始化失败” (常见坑)
PD26 有个很玄学的 Bug，有时候刚装好会提示连不上网。
如果遇到这种情况，不要慌，去 Mac 的 **终端 (Terminal)** 里输入这行命令重置一下网络权限：

```bash
sudo pkill -9 prl_client_app
```
输完密码回车，重启 PD 就行了。

### 第三步：安装 Windows 11
PD 最爽的一点就是不用你自己去满世界找 Windows 镜像。
1. 打开 PD，它会自动检测到你需要安装系统。
2. 直接点击 **“从 Microsoft 获取 Windows 11”** -> **“安装 Windows”**。
3. 喝杯咖啡，它会自动下载、安装 ARM 版的 Windows 11，全过程大概 10 分钟。

## ⚡️ 怎么设置最好用？ (防卡顿)

默认的配置其实有点保守，为了让体验更好，建议做这几个微调：

### 1. 内存分配
不要贪心！不要把 Mac 的内存全分给虚拟机。
* **8G 内存的 Mac**：分给 Windows **3GB** 就够了（保命要紧）。
* **16G 内存的 Mac**：分 **6GB** 最均衡。
* **配置入口**：右键 Dock 栏的 Windows 图标 -> 控制中心 -> 配置 -> 硬件 -> CPU与内存。

### 2. 融合模式 (Coherence)
这是 PD 的杀手锏。开启后，Windows 的任务栏会消失，Windows 软件就像 Mac 的原生软件一样漂浮在桌面上。
* **开启方式**：点击虚拟机左上角的蓝色小图标（或者菜单栏 -> 查看 -> 进入融合模式）。

### 3. 文件隔离 (强迫症必看)
默认情况下，Mac 的桌面文件会同步显示在 Windows 桌面上，乱得要死。
* **建议关闭**：配置 -> 选项 -> 共享 -> 取消勾选“与 Windows 共享 Mac 桌面”。

## 💬 常见问题 (FAQ)

**Q: M 芯片能装 Windows 10 吗？**
A: **极度不推荐。** 微软对 ARM 版 Win10 的支持已经停滞了，Win11 在 M 芯片上的效率是最高的，别去挑战兼容性。

**Q: 打游戏怎么样？**
A: PD26 对 DirectX 12 的支持好了很多。像《英雄联盟》、《原神》这种网游，在中画质下是可以流畅跑的。但如果你想跑 3A 大作……放过你的 Mac 吧，它真的尽力了。

**Q: 只有 Intel 架构的软件怎么办？**
A: Win11 ARM 版自带转译器，大部分 32位/64位 的 exe 软件都能直接运行，不用担心。
